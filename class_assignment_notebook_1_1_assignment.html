Detailed Description

Represents a problem (or Assignment) in an AssignmentNotebook.

Parameters

PROBLEM_Cells : [dict] This is the list of all notebook cells that has the tag PROBLEM. This is the problem description and solution skeleton. SOLUTION_Cells : [dict] This is the list of all notebook cells that has the tag SOLUTION. These are the proposed solution cells. Test_Cells : [dict] This is the list of all notebook cells that has the tag Test. These are the self tests (local tests) TEST_Cells : [dict] This is the list of all notebook cells that has the tag TEST. These are the final tests for the assignment, there should actually only be one.

Methods

amITEST() returns if this is a TEST only assignment. I.e. there are no, problem,solution,Test cells but only TEST cells.
